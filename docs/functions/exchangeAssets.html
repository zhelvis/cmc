Function exchangeAssets

Exchange Assets Returns the exchange assets in the form of token holdings. This information includes details like wallet address, cryptocurrency, blockchain platform, balance, and etc.

  • Only wallets containing at least 100,000 USD in balance are shown
  • Balances from wallets might be delayed
